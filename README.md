 🎓 LUCY STUDENT PORTAL

Welcome to the official repository for the **Lucy Student Portal**. This platform is designed to streamline the academic experience by providing students with a centralized hub to manage their school activities, track progress, and access essential resources.

---

 🌟 SYSTEM OVERVIEW

The portal serves as a digital bridge between students and the educational center. It focuses on usability, speed, and real-time data access.

➜ **Personalized Dashboard:** Upon login, students are greeted with a customized profile view including their photo and basic info.
➜ **Secure Authentication:** Integrated with Firebase to ensure user data protection and secure access.
➜ **Academic Tracking:** Real-time access to grades and attendance records.
➜ **Resource Management:** A dedicated section for downloading educational materials and assignments.
➜ **Communication Hub:** Instant access to school notices, schedules, and emergency contacts.

---

 🛠️ CORE TECHNOLOGY STACK

This project was built using modern web standards to ensure cross-device compatibility:

➜ **HTML5 & CSS3:** For a responsive and clean user interface.
➜ **JavaScript (ES6+):** Handling dynamic content and portal logic.
➜ **Firebase:** Managing User Authentication and Database services.
➜ **Font Awesome:** Providing intuitive iconography for better navigation.

---

 📂 NAVIGATION MODULES

The portal is divided into several easy-to-navigate modules:

➜ **Grades & Attendance:** Monitor academic performance and presence.
➜ **Assignments & Resources:** Stay updated with homework and study guides.
➜ **Schedule:** Never miss a class with the integrated timetable.
➜ **Social & Community:** Explore school clubs and the student gallery.
➜ **Emergency:** Quick-access contacts for student safety.

---
 🚀 HOW TO DEPLOY LOCALLY

Follow these steps to run the portal on your machine:

1. **Clone the Project:**
   `git clone https://github.com/etsubgezhagne-tech/Lucy-edu-website.git`

2. **Open the Entry Point:**
   Locate and open the `index.html` file in any modern web browser.

3. **Firebase Configuration:**
   Ensure your Firebase API keys are correctly set up in the script files for the Login/Register features to function.
